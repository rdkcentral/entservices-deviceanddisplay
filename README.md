## Warehouse Plugin Migration

**Note**: The Warehouse plugin has been migrated to its own standalone repository as of February 3, 2026.

- **New Repository**: [entservices-warehouse](https://github.com/rdkcentral/entservices-warehouse)
- **Reason**: Better modularity and independent versioning
- **Impact**: No functional changes; Warehouse continues to work as before
- **For existing deployments**: Update your build scripts to include both repositories if Warehouse is needed
